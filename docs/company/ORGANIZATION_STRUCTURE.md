# 1. Ce este Organization Structure

Organization Structure reprezintă modul în care WAP organizează și înțelege structura unei companii.

Scopul său nu este doar reprezentarea unei organigrame.

Scopul este înțelegerea relațiilor dintre companie, locații, departamente, echipe, roluri și angajați.

O structură bine definită permite Wapy să ofere recomandări mai precise, să analizeze nevoile companiei și să sprijine dezvoltarea acesteia pe termen lung.

Organization Structure reprezintă fundația tuturor funcțiilor dedicate companiilor din ecosistemul WAP.
# 2. Structura organizației

O organizație poate fi formată din una sau mai multe locații.

Fiecare locație poate avea unul sau mai multe departamente.

Fiecare departament poate avea una sau mai multe echipe.

Fiecare echipă poate avea unul sau mai multe roluri.

Fiecare rol poate fi ocupat de unul sau mai mulți angajați.

Această structură permite Wapy să înțeleagă modul în care funcționează organizația și să analizeze atât compania în ansamblu, cât și fiecare nivel al acesteia în mod individual.
