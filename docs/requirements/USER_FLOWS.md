# 1. Scop

Acest document descrie pașii prin care participanții folosesc WAP în situații reale.

Scopul nu este descrierea tehnică a sistemului, ci înțelegerea experienței utilizatorului de la primul contact cu platforma până la obținerea unui rezultat concret.

Fiecare flux trebuie să fie simplu, clar și orientat către rezolvarea unei probleme reale.
# 2. Fluxul muncitorului nou

## Obiectiv

Un muncitor care utilizează pentru prima dată WAP trebuie să își poată crea profilul profesional și să primească primele recomandări relevante într-un timp cât mai scurt.

## Pași

1. Muncitorul deschide WAP.
2. Alege rolul de muncitor.
3. Creează un cont sau se autentifică.
4. WAP explică pe scurt beneficiile platformei.
5. Muncitorul completează informațiile esențiale.
6. Wapy începe analiza profilului.
7. Wapy identifică primele oportunități relevante.
8. Muncitorul primește recomandările și poate continua dezvoltarea profilului.

## Rezultat

Muncitorul trebuie să înțeleagă rapid valoarea platformei și să poată începe dezvoltarea carierei fără procese complicate.
# 3. Prima experiență

Prima experiență a utilizatorului este una dintre cele mai importante etape ale platformei.

În primele minute utilizatorul trebuie să înțeleagă:

- ce este WAP;
- ce poate face WAP pentru el;
- ce trebuie să completeze;
- ce beneficii primește imediat.

Wapy trebuie să ofere valoare încă din primele minute de utilizare.

Scopul este ca utilizatorul să simtă că platforma îl ajută încă de la prima utilizare.
