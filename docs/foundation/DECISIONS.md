# DECISION-001

## Titlu
Muncitorii nu plătesc pentru a-și căuta un loc de muncă.

## Status
Approved

## Data
2026-07-03

## Motiv
Protejează ecosistemul și încurajează utilizarea platformei.

## Impact
- Crește încrederea utilizatorilor.
- Îmbunătățește ecosistemul.
- Monetizarea se mută către servicii și automatizări pentru companii.
