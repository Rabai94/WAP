# WAP Design

Acest folder centralizeaza materialele de design pentru proiectul WAP.

## Structura

- `01_Brand_DNA/` - principii de brand, directie vizuala, ton, culori si repere conceptuale.
- `02_Hero/` - imagini, compozitii si variante pentru zonele hero ale aplicatiei sau materialelor de prezentare.
- `03_Backgrounds/` - fundaluri, texturi si elemente vizuale de ambient.
- `04_Icons/` - iconite si seturi de pictograme folosite in interfata sau materiale.
- `05_Logos/` - logo-uri, variante de logo si fisiere sursa asociate.
- `06_Badges/` - insigne, etichete vizuale, markere si elemente de status.
- `07_UI/` - componente de interfata, explorari de layout si resurse pentru designul produsului.
- `08_Animations/` - animatii, motion studies, prototipuri si fisiere aferente.
- `09_References/` - referinte vizuale, inspiratie, moodboards si materiale externe de lucru.
- `10_Exports/` - fisiere exportate si livrabile pregatite pentru folosire sau distribuire.
